GestureController
=================

iOS/Android application that uses gestures to control audio effects


Prospective Effects:

Phase 1:
  Tremolo (LFO)
  Delay (IIR / FIR)
  Vibrato (LFO)

Phase 2:
  Wah (IIR / FIR)
  Reverb
  Granular Synthesis
  
Phase 3:
  Pitch Shifting
  Overdrive / Distortion
